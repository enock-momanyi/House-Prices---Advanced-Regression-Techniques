# House-Prices---Advanced-Regression-Techniques
Uses Linear Regression to predict House Prices

From https://www.kaggle.com/c/house-prices-advanced-regression-techniques

The notebook `HousePrice.ipynb` contains my implementation which included:
- Data Visualization
- Feature engineering(log transformation)
- Dummy variable creation for categorical variables
- etc.

The simple linear model achieves a score of `0.13412` on Kaggle
